# Planetfall
Игрок — космонавт, чей корабль потерпел крушение на одной из планет неизученной звёздной системы. Чтобы починить корабль и отправиться домой, ему нужно собрать детали корабля, разлетевшиеся по разным планетам этой системы. Каждая планета — это новый уровень с уникальной врагамии препятствиями.
# Техническое задание
Игра будет включать в себя несколько уровней, выбора персонажа, кастомизацию, меню и базовую механику плаформера и основана на Python с использованием библиотеки Arcade, вдохновлена играми серии Mario с космической тематикой.
## Концепция
Игрок управляет космонавтом, который сражается с врагами, исследует разные планеты и перемещается по ним с целью собрать детали разрушенной ракеты для того, чтобы вернуться на свою планету. Планируется сделать главное меню(начальный экран, выбор и кастомизация персонажа, выход), меню выбора уровня(доступные и недоступные), меню паузы и окончания игры.
