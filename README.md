# Planetfall
Игрок — космонавт, чей корабль потерпел крушение на одной из планет неизученной звёздной системы. Чтобы починить корабль и отправиться домой, ему нужно собрать детали корабля, разлетевшиеся по разным планетам этой системы. Каждая планета — это новый уровень с уникальными врагами и препятствиями.
# Техническое задание
Игра будет включать в себя несколько уровней, выбор персонажа, кастомизацию, меню и базовую механику плаформера, и будет основана на Python с использованием библиотеки Arcade, вдохновлена играми серии Mario, только с космической тематикой.
## Концепция
Игрок управляет космонавтом, который сражается с врагами, исследует разные планеты и перемещается по ним с целью собрать детали разрушенной ракеты для того, чтобы вернуться на свою планету. Планируется сделать главное меню(начальный экран, выбор и кастомизация персонажа, выход), меню выбора уровня(доступные и недоступные), меню паузы и окончания игры.
# Пояснительная записка
Для установки всех нужных библиотек введите в терминал:
```shell
pip install -r requirements.txt
```
PlanetFall является 2D-платформером. При запуске появляется "заставка" - краткая информация о том, как персонаж оказался в данном месте. После запускается главное меню(выбор персонажа и уровня). Уровни и собранные на уровнях элементы сохраняются в базе данных. При наличии подходящей базы данных, данные берутся из неё, при отсутствии - создаётся новая, и все следующие данные записываются в неё.
При прохождении уровней игрок должен собирать предметы - части разбившегося корабля - и избегать врагов и смертельных препятсвий. После прохождения уровня высвечивается страница с количеством собранных элементов и временем прохождения, возможностью перепрохождения уровня, перехода к следующему уровню и выходу в меню. После прохождения всех уровней выводится количество собранных элементов за последний уровень, все уровни, время прохождения, и появляется возможность перепройти уровень, выйти в меню или сбросить прогресс(то есть начать с начала).
Упраление осуществляется кнопками AD/←→ для передвижения назад и вперёд, SPACE для прыжка и ESC для паузы/выхода в меню.
