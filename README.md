# Planetfall
Игрок — космонавт, чей корабль потерпел крушение на одной из планет неизученной звёздной системы. Чтобы починить корабль и отправиться домой, ему нужно собрать детали корабля, разлетевшиеся по разным планетам этой системы. Каждая планета — это новый уровень с уникальной врагамии препятствиями.
